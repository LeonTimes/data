## 三、数据获取详细说明

### D1：AI岗位占比

```
数据来源选项：
1. 智联招聘年度报告（需购买或联系平台获取省级数据）
2. Boss直聘研究院年度报告（公开部分数据）
3. 中国就业研究所年度报告

替代方案实现：
指标名称：信息传输、软件和信息技术服务业从业人员占比
计算公式：从业人员占比 = IT服务业从业人员数 / 全省从业人员总数
数据来源：各省统计年鉴「从业人员」章节
时间跨度：2011-2022年（需手动录入）
```

### D2：AI投融资规模

```
数据来源选项：
1. IT桔子官网 → 按省份筛选 → 导出融资数据（需注册账号）
2. 私募通数据库（需购买）
3. 艾瑞咨询《中国人工智能产业研究报告》

替代方案实现：
指标名称：数字经济核心产业固定资产投资额
数据来源：各省统计年鉴「固定资产投资」章节
计算公式：投资额 = 信息传输软件IT服务业固定资产投资（亿元）
```

### D3：省级AI政策强度

```
数据来源：北大法宝/北大法意政策数据库
检索关键词："人工智能" + "省份名" + "年份"

构建方法：
1. 搜索各省2017-2023年AI相关政策文件
2. 按政策类型赋权：
   - 法律法规类（省级人大立法）：权重 = 3
   - 部门规章类（省政府/厅局文件）：权重 = 2
   - 规范性文件类（通知、意见、方案）：权重 = 1
3. 加权求和得到各省各年政策强度指数

替代方案实现（文本挖掘）：
数据来源：各省政府工作报告（2017-2023年）
方法：提取工作报告全文，统计"人工智能"关键词出现频次
Python代码示例：
```python
import re
text = open('政府工作报告.txt').read()
count = len(re.findall('人工智能', text))
```

### D4：AI试验区虚拟变量

```
工信部AI创新发展试验区名单（截至2021年）：

2019年首批：北京、上海
2020年第二批：杭州、广州、深圳、天津、长沙、合肥
2021年第三批：成都、重庆、西安、武汉、南京、苏州、济南

虚拟变量设置规则：
设立当年及以后 = 1，设立前 = 0，非试验区省份 = 0

示例：
北京：2019-2023 = 1，2011-2018 = 0
杭州：2020-2023 = 1，2011-2019 = 0
成都：2021-2023 = 1，2011-2020 = 0
其他省份：全部 = 0
```

### D5：东中西部分组代码

```
国家统计局官方分组标准：

东部地区（10省）：
北京、天津、河北、上海、江苏、浙江、福建、山东、广东、海南

中部地区（6省）：
山西、安徽、江西、河南、湖北、湖南

西部地区（12省）：
内蒙古、广西、重庆、四川、贵州、云南、西藏、陕西、甘肃、青海、宁夏、新疆

东北地区（3省）：
辽宁、吉林、黑龙江

分组代码建议：
东部 = 1，中部 = 2，西部 = 3，东北 = 4
```

### D6：地形起伏度/沿海虚拟变量

```
地形起伏度计算：
公式：起伏度 = (最高海拔 - 最低海拔) / 省域面积
数据来源：DEM数字高程模型数据（地理空间数据云）

替代方案（简化）：
各省平均海拔数据（公开资料）

沿海虚拟变量设置：
沿海省份 = 1：辽宁、河北、天津、山东、江苏、上海、浙江、福建、广东、广西、海南
内陆省份 = 0：其他14省
```
